# Implementation Plan: Mateus Felipe Portfolio

## 1. Design Concept: "The Digital Architect"
- **Theme**: Dark, immersive, futuristic but clean.
- **Colors**:
  - Background: Deep Void (`#0a0a0a`) to Midnight Blue (`#0f172a`).
  - Accents: Neon Cyan (`#00f3ff`), Electric Purple (`#bd00ff`), Bright White (`#ffffff`).
- **Typography**: Modern Sans-Serif (e.g., 'Inter' or 'Outfit').
- **Vibe**: High-performance, precise, artistic.

## 2. Structure
- **Hero Section**:
  - Headline: "Building Digital Experiences".
  - Subheadline: "Mateus Felipe | Full-Stack Developer".
  - Visual: CSS-based particle or gradient mesh animation.
- **About Section**:
  - Minimalist bio with a "terminal" code snippet aesthetic.
- **Projects Gallery**:
  - Horizontal scroll or Grid with glassmorphism cards.
  - Hover effects revealing tech stack.
- **Skills**:
  - Animated floating tags or progress bars.
- **Contact**:
  - Simple, elegant form with validation animations.

## 3. Tech Stack
- **Framework**: Next.js 15 (App Router).
- **Styling**: Vanilla CSS (CSS Modules & Variables) for maximum control over animations.
- **Icons**: Lucide React or similar.

## 4. Immediate Steps
1.  Clean up existing Next.js boilerplate.
2.  Define CSS Variables in `globals.css`.
3.  Create the `Hero` component with the "wow" factor.
